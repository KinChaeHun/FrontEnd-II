# FrontEnd-II

1. 프로젝트 소개

본 프로젝트는 영어 문장을 단어 순서대로 완성하는 웹 기반 게임이다.
사용자는 제시된 한국어 문장을 보고, 아래에 제공된 영어 단어 카드를
드래그 앤 드롭 방식으로 순서에 맞게 배치하여 문장을 완성한다.

게임 시작 시 사용자 이름을 입력받아
사용자별 점수 및 게임 진행 이력을 저장·관리하도록 구현하였다.

2. 주요 기능
2.1 사용자 관리

게임 시작 전 이름 입력 화면 제공
이름을 기준으로 사용자 데이터 생성
localStorage를 활용하여 사용자별 점수 및 기록 저장
새로고침 후에도 이전 기록 유지

2.2 게임 진행

한국어 문장 제시
영어 문장을 구성하는 단어 수만큼 빈칸 자동 생성
단어 카드를 드래그 앤 드롭하여 빈칸에 배치
빈칸은 하나당 하나의 단어만 배치 가능
슬롯에 들어간 단어는 다시 카드 영역으로 이동 가능

2.3 점수 시스템

정답일 경우
→ 단어 1개당 5점
오답일 경우
→ 점수 없음
누적 점수 실시간 표시

2.4 게임 종료 처리

모든 문제를 풀면
→ “모든 문제가 끝났습니다” 메시지 출력
이후 정답 확인 및 다음 문제 버튼 비활성화
게임이 자동으로 다시 시작되지 않음

2.5 게임 기록 관리

문제별 결과 저장 (정답 / 오답)
문제별 획득 점수 기록
사용자별 게임 이력 목록 출력

3. 사용 기술

HTML5
CSS3
JavaScript (ES6)
jQuery
HTML5 Drag & Drop API
localStorage

4. 실행방법

프로젝트 폴더를 다운로드
로컬 서버 실행 (예: Python)
"python -m http.server 5500"

브라우저에서 접속
"http://localhost:5500"
